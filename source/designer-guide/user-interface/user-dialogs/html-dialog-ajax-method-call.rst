Ajax Method Call API
--------------------

Methods of an HTML Dialog can be called with JavaScript through a REST-like
interface. You can use this Ajax method call API of Axon Ivy to integrate
JavaScript libraries like D3, jQuery or your own JavaScript frameworks. See the
Ajax Method Call Demo in the HTML Dialog Demo Project:
